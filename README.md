# 🧑‍💼 Entrevistador de Carreira em Tecnologia

Este projeto documenta a interação inicial com o entrevistador especializado em descobrir o perfil profissional de pessoas interessadas em tecnologia.

## 🎯 Objetivo
Conduzir uma entrevista estruturada de 7 perguntas para entender:
- Interesses e motivações  
- Experiência prévia  
- Disponibilidade de estudo  
- Preferências de trabalho  
- Objetivos profissionais  

## 📝 Estrutura da Entrevista
1. O que mais atrai em tecnologia (resolver problemas, criar produtos ou entender sistemas).  
2. Experiência prévia na área.  
3. Quantas horas semanais disponíveis para estudo.  
4. Preferência no dia a dia (pessoas, dados ou código).  
5. Objetivo principal (primeiro emprego, transição ou crescimento).  
6. Interesses técnicos (web, dados, IA, infraestrutura).  
7. Experiência prévia aproveitável.  

## 📊 Resultado da Entrevista
Após as 7 perguntas, foram sugeridas 3 carreiras ranqueadas:

1. **Cientista de Dados / Engenheiro de Dados** – 18/20  
2. **Engenheiro de Machine Learning / IA** – 17/20  
3. **Engenheiro de Automação / Integração de Sistemas** – 16/20  

O usuário escolheu seguir com a **3ª opção: Engenheiro de Automação / Integração de Sistemas**.

---

## 📈 Resultados da Fase 1
- Entrevista concluída com sucesso em 7 perguntas.  
- Carreira escolhida: **Engenheiro de Automação / Integração de Sistemas**.  
- Informações coletadas foram transferidas para o planejador especializado (Agent 2).

# 📚 Planejador de Roadmap de Carreira em Tecnologia

Este projeto documenta a interação com o planejador especializado em criar roadmaps personalizados de carreira em tecnologia.

## 🎯 Objetivo
Gerar um plano completo de estudos para a carreira escolhida:
- **Engenheiro de Automação / Integração de Sistemas**

## 📥 Dados Recebidos
- **Carreira escolhida:** Engenheiro de Automação / Integração de Sistemas  
- **Horas semanais:** 16h  
- **Experiência:** iniciante com base prática (n8n, APIs, lógica de sistemas, agentes LLM)  
- **Objetivo:** transição de carreira  
- **Preferência:** dados  
- **Interesses:** inteligência artificial, infraestrutura, dados e desenvolvimento web  

---

## 📦 Plano Gerado

### 🧩 Visão do Dia a Dia
- Criar e manter fluxos de automação  
- Integrar APIs  
- Configurar e monitorar ferramentas de orquestração  
- Garantir escalabilidade e segurança  
- Documentar integrações  

### 🧠 Mapa de Skills
**Core Skills:** lógica de programação, APIs REST, automação de processos  
**Nice-to-Have:** infraestrutura, IA aplicada  
**Ferramentas:** n8n, Zapier, Python, Docker  

### 📅 Roadmap de 90 Dias
- **Mês 1:** Fundamentos (JSON, APIs, automações simples)  
- **Mês 2:** Prática (SQL, Docker, integrações complexas)  
- **Mês 3:** Portfólio e preparação (projeto completo, entrevistas, publicação no GitHub)  

### 🚀 Projeto de Portfólio
**Projeto:** Plataforma de Automação de Atendimento  
**Entregáveis:** fluxo funcional, documentação, vídeo demonstrativo  

### 💬 Roteiro de Entrevistas
5 perguntas comuns com respostas estruturadas (ex.: experiência com automação, integração de sistemas, segurança).  

### 🎓 Trilha DIO Recomendada
**Formação Automação de Processos com n8n**  

---

## 📈 Resultados da Fase 2
- Roadmap de 90 dias criado e adaptado para 16h/semana.  
- Projeto de portfólio definido para validação prática.  
- Roteiro de entrevistas estruturado para preparação.  
- Trilha DIO recomendada para complementar estudos.

# ✅ Conclusão Geral

A jornada foi dividida em duas fases:  
1. **Entrevista** para identificar perfil e carreira ideal.  
2. **Planejamento** com roadmap de estudos, projeto de portfólio e preparação para entrevistas.  

O caminho escolhido foi **Engenheiro de Automação / Integração de Sistemas**, com plano estruturado para transição de carreira e prática consistente em 16h semanais.
